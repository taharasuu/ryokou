#ryokou 

## サイト概要
### サイトテーマ
旅行好きのためのSNS

### テーマを選んだ理由
旅行が好きで良くSNSやサイトで観光地や食べ物を探しているが、
詳しい情報や実際に行ってどうだったかなどの評価があったらいいなと思った。
また、穴場スポットを旅行好き同士でシェアしたいと思った

### ターゲットユーザ
・旅行好き

### 主な利用シーン
旅行に行く時に行きたい施設や観光地を探す、シェア
